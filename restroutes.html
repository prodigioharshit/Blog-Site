RESTful Routes

A Table of all 7 RESTful Routes
Name Path HTTP Verb Purpose Mongoose Method
Index /dogs GET List All dogs Dog.find()--where Dog is mongoose model variable
New /dogs/new GET Show new Dog form N/A
Create /dogs POST Create a new Dog and redirect somewhere Dog.create()
Show /dogs/:id GET Show info about one specific dog Dog.findById()
Edit /dogs/:id/edit GET Show edit form of one dog Dog.findById()
Update /dogs/:id PUT Update particular Dog then redirect somewhere Dog.findByIdAndUpdate()
Destroy /dogs/:id DELETE Delete a particular dog then redirect somewhere Dog.findByIdAndRemove()
